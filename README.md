# Topdeck Duel — Legal Portal

This repository is used to host static web pages containing legal documents for the card game **Topdeck Duel** (Developer: **One Ink Studio**).

The site is deployed using GitHub Pages. These files are required to comply with moderation and publishing policies on the Google Play Store.

## Links for Google Play Console

Direct links to insert into the "App Content" section of the Google Play Developer Console:

* **Home Page:** [https://vladislavpetukhov.github.io/](https://vladislavpetukhov.github.io/)
* **Privacy Policy:** [https://vladislavpetukhov.github.io/privacy-policy](https://vladislavpetukhov.github.io/privacy-policy)
* **Terms & Conditions:** [https://vladislavpetukhov.github.io/terms-and-conditions](https://vladislavpetukhov.github.io/terms-and-conditions)

*(Note: If your GitHub Pages setup requires file extensions, you may need to append `.html` to the ends of the Privacy Policy and Terms & Conditions URLs).*

## File Structure

* `index.html` — Landing page with navigation links.
* `privacy-policy.html` — Privacy Policy detailing data collection and usage practices (including third-party services).
* `terms-and-conditions.html` — Terms and Conditions for using the application.
